# MySQL Connector/J

Download `mysql-connector-j-9.x.x.jar` dari https://dev.mysql.com/downloads/connector/j/ dan letakkan di folder ini.

Atau jika pakai Maven, cukup tambahkan dependency:
```xml
<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <version>9.x.x</version>
</dependency>
```

**Untuk proyek ini (manual):**
1. Download JAR dari https://dev.mysql.com/downloads/connector/j/
2. Pilih "Platform Independent" → ZIP
3. Extract, cari file `mysql-connector-j-9.x.x.jar`
4. Copy ke folder `lib/`
